# Foodpalace
it a simple food ordering app. It is a serveless app.
## Technologies
- React Native
- Expo cli
- Firebase

## Features
- Login
- Register
- Browser Food
- View food advertisement banner
- Choose foot categories
- like Food
- View Food details
- Add food quantity
- Add food to cart
- View Cart items
- Remove from cart
- Checkout cart
- Logout

## More Info
- The app  is responsive
- The app was not tested on IOS phone
- Easy to use
